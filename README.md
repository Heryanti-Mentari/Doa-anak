\# Website Doa dan Kuis Islami Anak Berbasis Kecerdasan Buatan



Website edukasi berbasis web yang dirancang untuk membantu anak-anak

mengenal doa harian dan pengetahuan Islami melalui kuis interaktif.

Website ini memanfaatkan konsep kecerdasan buatan sederhana untuk

meningkatkan interaksi pengguna.

\## Tujuan

\- Menyediakan media pembelajaran Islami yang interaktif untuk anak-anak  

\- Menerapkan konsep kecerdasan buatan dalam sistem kuis edukatif  

\- Meningkatkan minat belajar anak melalui audio dan interaksi dinamis  



\## Konsep Kecerdasan Buatan

Website ini menerapkan konsep kecerdasan buatan sederhana berupa:

\- Pemilihan soal kuis secara dinamis

\- Evaluasi jawaban pengguna secara otomatis

\- Pemberian umpan balik (feedback) berdasarkan jawaban pengguna



\## Teknologi yang Digunakan

\- HTML

\- CSS

\- JavaScript



\## Fitur Utama

\- Kumpulan doa harian anak

\- Kuis Islami interaktif

\- Audio bacaan doa

\- Antarmuka ramah anak
